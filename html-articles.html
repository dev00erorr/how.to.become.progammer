اساسيات HTML

مقدمة في HTML

HTML (HyperText Markup Language) هي اللغة الأساسية لبناء صفحات الويب. تُستخدم لوصف هيكل الصفحات باستخدام مجموعة من الوسوم (Tags) التي تُحدد أجزاء الصفحة مثل العناوين، الفقرات، الروابط، الصور، والجداول.

وظائف HTML

  • إنشاء هيكل صفحات الويب: تُستخدم HTML لتحديد العناصر المختلفة في صفحة الويب مثل النصوص، العناوين، الصور، والروابط.
  • دمج العناصر المتعددة: تساعد HTML في دمج الوسائط مثل الصور والفيديوهات في الصفحات.
  • التفاعل مع CSS وJavaScript: HTML تعمل مع CSS لتنسيق الصفحات وJavaScript لإضافة التفاعلية.

هيكل صفحة HTML

هيكل صفحة HTML يتكون من:

  • وسم : يحدد نوع المستند.
  • وسم : يمثل الجذر الذي يحتوي على جميع عناصر الصفحة.
  • وسم : يحتوي على معلومات وصفية عن الصفحة مثل العنوان وترميز اللغة.
