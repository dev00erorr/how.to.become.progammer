اساسيات C++

مقدمة في C++

C++ هي لغة برمجة متعددة الأغراض، تُستخدم بشكل واسع في تطوير البرمجيات، الألعاب، والأنظمة المضمنة. تم تصميمها كامتداد للغة C، وتضيف ميزات مثل البرمجة الكائنية والتخصيص الديناميكي للذاكرة.

وظائف C++

  • تطوير البرمجيات: تُستخدم C++ لتطوير البرمجيات الكبيرة والمعقدة.
  • برمجة الألعاب: تعتبر C++ من اللغات الرائدة في صناعة الألعاب.
  • التعامل مع الأنظمة المضمنة: تُستخدم C++ في تطوير الأنظمة التي تحتاج إلى التحكم في العتاد مباشرة.

ميزات C++

تتضمن C++ العديد من الميزات، منها:

  • البرمجة الكائنية (OOP): تدعم C++ مبادئ OOP مثل الكائنات، الفئات، والوراثة.
  • التخصيص الديناميكي للذاكرة: يسمح للمطورين بإدارة الذاكرة بشكل أكثر كفاءة.
  • قابلية النقل: يمكن تشغيل برامج C++ على أنظمة تشغيل متعددة دون تعديل كبير.

أنواع البيانات الأساسية في C++

  • int: للأعداد الصحيحة.
  • float: للأعداد العشرية.
  • double: للأعداد العشرية ذات الدقة العالية.
  • char: لتخزين حرف واحد.
  • bool: لتخزين القيم الصحيحة (true أو false).

مثال بسيط على برنامج C++

                #include <iostream>
                using namespace std;

                int main() {
                    cout << "مرحباً بك في C++" << endl;
                    return 0;
                }

الكود أعلاه يقوم بعرض عبارة "مرحباً بك في C++" على الشاشة باستخدام دالة cout.
