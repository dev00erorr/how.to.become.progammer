اساسيات Python

مقدمة في Python

Python هي لغة برمجة عالية المستوى، متعددة الأغراض وسهلة التعلم. تُستخدم بشكل واسع في تطوير الويب، تحليل البيانات، الذكاء الاصطناعي، وتعلم الآلة.

وظائف Python

  • تطوير تطبيقات الويب: تُستخدم Python مع إطارات مثل Django وFlask لبناء تطبيقات ويب ديناميكية.
  • تحليل البيانات: تُستخدم Python مع مكتبات مثل Pandas وNumPy لتحليل البيانات ومعالجتها.
  • تطوير الذكاء الاصطناعي: تُستخدم Python بشكل واسع في مجالات تعلم الآلة والذكاء الاصطناعي باستخدام مكتبات مثل TensorFlow وKeras.

ميزات Python

تتضمن Python العديد من الميزات، منها:

  • **بسيطة وسهلة القراءة**: تُعتبر Python من أسهل اللغات في التعلم، مما يجعلها مثالية للمبتدئين.
  • **تفاعلية**: يمكن استخدام Python في بيئات تفاعلية، مما يسهل اختبار التعليمات البرمجية.
  • **مكتبات غنية**: تحتوي Python على مجموعة كبيرة من المكتبات والأطر التي تسهل تطوير التطبيقات المختلفة.

أنواع البيانات الأساسية في Python

  • **int**: للأعداد الصحيحة.
  • **float**: للأعداد العشرية.
  • **str**: لتخزين سلسلة من الأحرف.
  • **bool**: لتخزين القيم الصحيحة (True أو False).
  • **list**: لتخزين مجموعة من العناصر.
  • **dict**: لتخزين أزواج من المفاتيح والقيم.

مثال على برنامج Python

إليك مثالاً بسيطاً لبرنامج مكتوب بلغة Python يقوم بجمع عددين وإظهار النتيجة:


# طلب إدخال العدد الأول من المستخدم
num1 = float(input("أدخل العدد الأول: "))

# طلب إدخال العدد الثاني من المستخدم
num2 = float(input("أدخل العدد الثاني: "))

# حساب مجموع العددين
sum = num1 + num2

# عرض النتيجة
print("مجموع", num1, "و", num2, "هو:", sum)
